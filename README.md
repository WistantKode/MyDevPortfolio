## Introduction

This repository hosts the source code for Wistant Kode's personal portfolio website. Designed as a modern, responsive, and highly interactive platform, it serves to showcase my expertise as a DevSecOps Practicer and Software Engineer. The portfolio highlights a curated selection of projects, technical skills, and professional experiences, reflecting a commitment to robust backend architecture, scalable web solutions, and secure development practices.

My work is driven by a passion for Cloud Computing, Automation, and Cybersecurity, aiming to transform complex ideas into reliable and high-performance software.

## Features

- **Modern & Responsive UI/UX**: Crafted with a clean, intuitive design that adapts seamlessly across all devices.
- **Dynamic Hero Section**: Engaging introduction with a typewriter effect for key roles and interactive, animated technology icons.
- **Project Showcase**: Filterable and paginated display of projects, detailing technologies used, live demos, and code repositories.
- **Comprehensive Skillset**: A visually appealing section highlighting core technical competencies and tools.
- **Professional Journey Timeline**: A dual-column timeline presenting academic background and professional experiences.
- **Integrated Contact Form**: A functional contact form powered by Formspree for direct inquiries.
- **Theme Switching**: User-friendly light/dark mode toggle for personalized viewing experience.
- **Optimized SEO**: Strategically integrated keywords and meta descriptions for improved search engine visibility.
- **Senior Tone & Branding**: All content is tailored to reflect a senior, technical, and assertive professional persona, consistently branded as 'Wistant Kode'.

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

Ensure you have the following installed:
- [Node.js](https://nodejs.org/en/download/) (LTS version recommended)
- [npm](https://www.npmjs.com/get-npm) or [Yarn](https://yarnpkg.com/)
- [Git](https://git-scm.com/downloads)

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/wistantkode/your-portfolio-repo.git
    cd your-portfolio-repo
    ```
    *(Replace `https://github.com/wistantkode/your-portfolio-repo.git` with your actual repository URL)*

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Install `shadcn/ui` components (if prompted during setup or if you add new ones):**
    ```bash
    npx shadcn-ui@latest add dropdown-menu # Example for dropdown-menu
    # Repeat for any other shadcn/ui components used
    ```

### Running the Project

To start the development server:

```bash
npm run dev
# or
yarn dev
```

The application will typically be available at `http://localhost:5173` (or another port if 5173 is in use).
